# Домашнее задание № 3

## Задача № 1, 2 балла
Вспомним распределённый алгоритм обеспечения причинного порядка. В нём мы после обработки каждого сообщения вынуждены были просматривать 
все сохранённые, но не обработанные сообщения, чтобы понять, можем ли мы обработать какое-то из них. Найдите способ оптимизировать это. Предложите
оптимизацию алгоритма, которая позволит просматривать не все сохранённые сообщения, а только некоторые из них.

*Подсказка: Наверное, вы хотите про каждое сообщения при получении понимать, чего нам не хватает для того, чтобы его обработать.*

## Задача № 2, 2 балла
Рассмотрим алгоритм Лампорта для обеспечения общего порядка. Пусть `N` процессов-отправителей только отправляют сообщения 
получателям, а другие `M` процессов-получателей только получают и обрабатывают сообщения, посланные отправителями (итого в кластере `N + M` процессов).

Мы хотим, чтобы каждый получатель обрабатывал сообщения от отправителей в одном и том же порядке, то есть мы хотим обеспечить общий порядок. Можно сделать
это с помощью алгоритма Лампорта, потратив `3 * (N + M - 1)` сообщение на одну пересылку. Предложите способ уменьшить это число, немного 
модифицировав алгоритм Лампорта.

*Подсказка: Можно уменьшить это число до 3 * (N - 1) + M, что полезно в случае, когда число получателей много больше числа отправителей.*
